# Aviary Manager

Supabase-backed aviary management app built with Next.js.

## Setup

0. Use Node.js 20 (for `nvm`, run `nvm use` in this repo).
1. Create a Supabase project.
2. Add `.env.local` with your Supabase URL and anon key.
3. Install dependencies.
4. Run the dev server.

## Commands

- `npm install`
- `npm run dev`
- `npm run build`
